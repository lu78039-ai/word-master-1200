# Word Master Premium - 國中英語 1200 單字挑戰

這是一個專為國中生設計的英語單字學習網頁程式，包含教育部制定的 1200 個核心單字，並提供多種練習模式來強化記憶。

## 🚀 線上試玩
您可以透過 [GitHub Pages](https://pages.github.com/) 輕鬆部署本專案，讓學生隨時隨地在手機或平板上練習。

## 🌟 核心功能
- **六大模式**：
  - `英翻中` / `中翻英`：建立雙向聯想。
  - `克漏字`：從句子語境中學習用法。
  - `拼字練習`：強化字母拼寫記憶。
  - `聽力測驗`：使用系統語音合成，練習聽力。
  - `單字閃卡`：快速瀏覽與複習。
- **自適應設計**：支援手機、平板與電腦，完美填滿螢幕。
- **深色模式**：支援護眼模式，長時間使用不疲勞。
- **離線數據**：所有單字數據存放在 `data.js`，載入快速，支援靜態部署。

## 📂 檔案結構
- `index.html`: 主程式與美化外觀 (Vanilla CSS + HTML5 + JavaScript)。
- `data.js`: 1200 個單字的資料庫。
- `README.md`: 使用說明。

## 🛠️ 如何開始
1. 將此儲存庫 (Repository) Clone 或下載到本地。
2. 直接點擊 `index.html` 即可在瀏覽器運行。
